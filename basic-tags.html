Basic Tags 101

Text types

This is a paragraph. You make it by writing <p>text</p>

The text can be written in italics or in bold or in code

This is a blockquote. You make it by writing <blockquote>text</blockquote>
This is a preformated text. You make it by writing <pre>text</pre>

Titles also get smaller...

...according to their hierarchy...

...Just increase the number next to h in your code.

With the <a></a> tag, you can refer to other pages of the document or to websites. Like Google!

You can also add images directly from the web using <img src="website or from file">
