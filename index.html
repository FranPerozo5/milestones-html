Before starting

  • Open terminal outside of VS Code.
  • Use cd to navigate between directories.
  • Use ls to list contents of a directory.
  • Use mkdir to create an empty directory.
  • Use touch to create an empty file (but make sure you navigate in the new director first by using cd).
  • Use code nameofthefile to open VS Code.
