<p> is the opening tag and </p> is the closing tag. Anything in between is written in paragraph format.

<p> is the opening tag and <p> is the closing tag. It has attributes and values.

Attributes are extra bits of information which can be added to a tag. The purpose of attributes is to add extra information to the tag which can also be used to style the element. Attributes always appear within an opening tag and the value of the attribute always appears in quote marks.

In the example above, there are 4 attributes:

  • <href> is used to specify the URL of the page that the link goes to. Each value is http://google.com.
  • <id> is used to specify a unique identidication for an HTML to facilitate styling in a style sheet or JavaScript. Its value is the-best-link.
  • <class> is used to specify one or more class names for an HTML to facilitate styling in a style sheet or JavaScript. The only difference between them is that “id” is unique in a page and can only apply to at most one element, while “class” selector can apply to multiple elements. Its value is banana.
  • <target> is used to specify where the linked document will open when the link is clicked. Its value is _blank, which mwans that it opens the link in a new window. More information in this attribute can be found here.
